⚙️ Cómo configurar los competidores

Edita este archivo con un editor de texto, busca el bloque COMPETIDORES al final y cambia los WCA IDs reales. El WCA ID es ese código que aparece en el perfil de cada uno, tipo 2020SANC01.

Si alguno aún no tiene WCA ID (no ha competido nunca), puedes dejarlo vacío y aparecerá como sin datos.

const COMPETIDORES = [
  { nombre: "Alex",   wcaId: "2020XXXX01" },
  { nombre: "Marcos", wcaId: "2020XXXX02" },
  { nombre: "Ele",    wcaId: "2020XXXX03" },
  { nombre: "Andy",   wcaId: "2020XXXX04" },
];
